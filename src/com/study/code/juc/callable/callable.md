# Callable接口
创建线程的多种方式
- 继承Thread类
- 实现Runnable接口
- Callable接口
- 线程池方式


Callable接口和Runnable接口的区别
- 是否有返回值
- 是否抛出异常
- 实现方法名不同,一个是run(),一个是call()
